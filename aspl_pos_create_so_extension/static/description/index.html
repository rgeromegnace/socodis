Odoo 11 Create sales order from Point of Sale with Extended Features

Create sales order from Point of Sale with Extended Features


Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Also check this module: Create sales order from Point of Sale

Features

  • Create Sale order from POS(Quotations, Confirm, Paid)
  • Partial Payment of Sale order
  • Edit Quotations
  • Add or Create Shipping address and invoice address
  • Add Sale order note
  • View sale order with searching and filtering functionality
  • User can show only own sales order or all orders based on user rights.
  • Create sale order in selected warehouse.

How to use

  • Configuration to perform sale order operation from POS & Load order of last N number of days


  • We can configure user rights here for show all users sales order or not.


  • To create Sale order from POS and View past sale orders.
  • "Create Sale Order" button appears as per the POS configuration.


  • This popup appears when user creating sale order, either sale order operation is Quotations or Confirm or paid.
  • Change order date, add shipping address or create new one.


  • Add invoice address or create new one.


  • Set note for Sale order.


  • Direct link to view latest created sale order.


  • Sale orders list
  • User can edit Quotations, Partial or full pay of Quotations or Confirm order
