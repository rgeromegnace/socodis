POS Product Stock

This Module is used to display stock in POS interface also the seller can select and display different type of stock like "Qty on hand", "Incoming Qty", "Outgoing Qty", "Qty Available" in POS and Allow/Deny POS sale Order when product is out of stock. You can display stock of All & Specific stock location in POS. Also Stock will be automatically sync when order get placed in POS.

Key Features:

  • Display Stock in POS interface.
  • Seller can select and display different stock type like "Qty on hand", "Incoming Qty", "Outgoing Qty", "Qty Available" in POS.
  • Allow/Deny POS sale Order when product is out of stock.
  • Seller can set the Unit, Deny order when quantity goes down to set unit.
  • Display of Different type of Error messages in POS interface based on stock Availibility.
  • Show stock of all stock location and show stock of only specific stock location on POS session.
  • Stock will be automatically sync when order get placed in POS.

Stock Will be automatically Updated on POS Product Screen, no need to reload the POS Screen Everytime After you placed an Order.


Stock Quantity Validation with "All Warehouse" Configuration. If You add quantity more than available quantity in Orderline for a particular product then Stock Qty Validation will be added.


Stock Quantity Validation with "Current Session Warehouse" Configuration. If You add quantity more than available quantity in Current Warehouse in Orderline for a particular product then Stock Qty Validation will be added.


Stock Quantity Validation will not be applied for "Consumable" or "Service" Product. Although You Add More Qty in the Orderline, it will go to the payment screen without validation.
